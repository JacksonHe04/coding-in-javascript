## 5. 最长回文子串
**难度**：中等  

**提示**：给你一个字符串 `s`，找到 `s` 中最长的 <code>回文</code> 子串。  

### 示例 1
- **输入**：`s = "babad"`
- **输出**：`"bab"`
- **解释**：`"aba"` 同样是符合题意的答案。  

### 示例 2
- **输入**：`s = "cbbd"`
- **输出**：`"bb"`  

### 提示
- \(1 \leq s.length \leq 1000\)
- `s` 仅由数字和英文字母组成
